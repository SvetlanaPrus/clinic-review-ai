# Clinic Review AI

AI service for analyzing clinic reviews.

## 🚀 What the project does

- Reads reviews from CSV
- Analyzes using OpenAI
- Generates a report

## 🛠 Technologies

- Python
- FastAPI
- OpenAI API

## ▶️ How to run

```bash
pip install -r requirements.txt
uvicorn python-service.app:app --reload
```
